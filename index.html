    Q&A



    • three main differences between Web 1.0, 2.0 and 3.0:
      1. Most of the Web 1.0 pages are static. that means that they are Read-Only pages that do not change at all.
        In Contrast, on Web 2.0 and 3.0 its different. On Web 2.0 most of the web pages are Read-Write
        while in Web 3.0 the web pages are portable and personal.
      2. On Web 1.0 the web is a "company focused". That means all the content on the web was from companies
        providing that content while the community consume it - one-to-many communication model.
        However, On Web 2.0 the communication model changed to be "Community-focued"
        when others can also provide content. On Web 3.0, it transformed to be
        "Individual-focused" when a lot of the content on the web is personalized for everyone.
      3. On Web 1.0 one can only get to a web page by knowing the specific URL address of the page.
        On Web 2.0 there is a search engine that can find web pages that has been crawled before.
        Web 3.0 enable a better search result by applying smarter techniques for search.
    • Use Case for "Recipe for cinnamon cake":
      • On Web 1.0 a specific URL address of the relevant content needs to be known.
      • On Web 2.0 we can search "Recipe for a cinnamon cake" and get web pages that the owner of the web page
        made sure that the words "Recipe", "cinnamon", "cake" are tagged
      • On Web 3.0 the search "Recipe for a connamon cake" in the search engine will get as a result
        web pages that contain semantic content to the search words by applying advanced techniques
